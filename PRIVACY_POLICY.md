# Privacy Policy

**Discord Token Login Extension**

_Last updated: December 17, 2025_

## 1. Data Collection
This extension does **not** collect, store, or transmit any personal data, Discord tokens, or user information to any server or third party. All data used by the extension remains local to your browser and device.

## 2. Permissions
The extension requests only the minimum permissions required to function (such as storage and tabs). These permissions are used solely for the operation of the extension and not for tracking or analytics.

## 3. Use of Discord Token
The Discord token you provide is used only to log you into your Discord account via the browser. The token is never sent to any external server or shared with anyone.

## 4. Third-Party Services
This extension does not use any third-party analytics, advertising, or tracking services.

## 5. Security
We take your privacy seriously. All operations are performed locally in your browser. We recommend never sharing your Discord token with anyone and only using this extension for your own accounts.

## 6. Changes to this Policy
Any changes to this privacy policy will be updated in this file and reflected in future versions of the extension.

## 7. Contact
If you have any questions or concerns about this privacy policy, please contact the developer via the contact links in the extension popup.

---

By using this extension, you agree to this privacy policy.
